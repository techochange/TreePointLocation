# TreePointLocation
三点定位的简易实现

这是一个三点定位算法的数学实现

实现思路是
采用圆在正交直角坐标系上的交点求解
```
            (x-a)^2+(y-b)^2=r^2
             =>x^2-2ax+a^2+y^2-2by+b^-r^=0
             =>x^2+y^2+(-2a)x+(-2b)y+(a^2+b^2-r^2)=0;
             =>x^2+y^2+Dx+Ey+F=0
             其中
             D=-2a
             E=-2b
             F=a^2+b^2-r^
             
             x^2+y^2+D1x+E1y+F1=0  (1)
             x^2+y^2+D2x+E2y+F2=0  (2)
             x^2+y^2+D3x+E3y+F3=0  (3)

            由(2)-(1)得
            (D2-D1)x+(E2-E1)y+F2-F1=0
            (D1-D2)x=(E2-E1)y+F2-F1
            x=((E2-E1)y+F2-F1)/(D1-D2)

            由(3)-(1)得
            (D3-D1)x+(E3-E1)y+F3-F1=0
            x=((E3-E1)y+F3-F1)/(D1-D3) = ((E2-E1)y+F2-F1)/(D1-D2)
            得
            ((E3-E1)y+F3-F1)*(D1-D2) = ((E2-E1)y+F2-F1)(D1-D3)
            (E3-E1)y(D1-D2)+(F3-F1)(D1-D2) = (E2-E1)y(D1-D3)+(F2-F1)(D1-D3)
            (E3-E1)(D1-D2)y-(E2-E1)(D1-D3)y = (F2-F1)(D1-D3)-(F3-F1)(D1-D2)
            得
            y= ((F2-F1)*(D1-D3)-(F3-F1)*(D1-D2))/((E3-E1)*(D1-D2)-(E2-E1)*(D1-D3))

            得
            x=((E3-E1)*y+F3-F1)/(D1-D3) 或 x=((E2-E1)*y+F2-F1)/(D1-D2)
```
对测量误差具有一定的容错性
